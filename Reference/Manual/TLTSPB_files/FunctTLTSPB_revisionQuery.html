TLTSPB_revisionQuery

ViStatus TLTSPB_revisionQuery (ViSession instrumentHandle, ViChar _VI_FAR instrumentDriverRevision[], ViChar _VI_FAR firmwareRevision[]);

Purpose

This function returns the revision numbers of the instrument driver and the device firmware.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session. You may pass VI_NULL.
Note:
If you pass VI_NULL an empty string will be returned for the firmware revision parameter.
Output
Name Type Description
instrumentDriverRevision ViChar[] This parameter returns the Instrument Driver revision.
Notes:
(1) The array must contain at least 256 elements ViChar[256].
(2) You may pass VI_NULL if you do not need this value.
firmwareRevision ViChar[] This parameter returns the device firmware revision.
Notes:
(1) The array must contain at least 256 elements ViChar[256].
(2) You may pass VI_NULL if you do not need this value.

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
