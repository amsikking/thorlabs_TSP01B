TLTSPB_getRsrcInfo

ViStatus TLTSPB_getRsrcInfo (ViSession instrumentHandle, ViUInt32 deviceIndex, ViChar _VI_FAR modelName[], ViChar _VI_FAR serialNumber[], ViChar _VI_FAR manufacturerName[], ViPBoolean resourceInUse);

Purpose

With this function, the user gets more information about the connected instruments that are found in the function findRsrc().

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
deviceIndex ViUInt32 Index of the connected device you want to get the resource name from (0 to number of connected devices - 1).
see TLTSPB_getDeviceCount()
Output
Name Type Description
modelName ViChar[] A 256 byte string buffer to get the model name of the connected device at the given index
serialNumber ViChar[] A 256 byte string buffer to get the serial number of the connected device at the given index
manufacturerName ViChar[] A 256 byte string buffer to get the manufacturer name of the connected device at the given index
resourceInUse ViBoolean (passed by reference) Boolean flag indicating if the device at given index is already in use by another instance

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
