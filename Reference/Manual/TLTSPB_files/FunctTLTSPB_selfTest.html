TLTSPB_selfTest

ViStatus TLTSPB_selfTest (ViSession instrumentHandle, ViPInt16 selfTestResult, ViChar _VI_FAR description[]);

Purpose

This function runs the device self test routine and returns the test result.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
Output
Name Type Description
selfTestResult ViInt16 (passed by reference) This parameter contains the value returned from the device self test routine. A retured zero value indicates a successful run, a value other than zero indicates failure.
description ViChar[] This parameter returns the interpreted code as an user readable message string.
Note:
The array must contain at least 256 elements ViChar[256].

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
