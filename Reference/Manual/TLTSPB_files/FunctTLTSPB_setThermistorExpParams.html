TLTSPB_setThermistorExpParams

ViStatus TLTSPB_setThermistorExpParams (ViSession instrumentHandle, ViUInt16 channel, ViReal64 r0Value, ViReal64 t0Value, ViReal64 betaValue);

Purpose

This function sets the thermistor exponential method parameters.
Note:
This function is valid for external temperature sensors only

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
channel ViUInt16 This parameter specifies the temperature channel.
Note:
This function is valid for all temperature sensors only (channels 1, 2 and 3)
r0Value ViReal64 This parameter specifies the R0 value.
t0Value ViReal64 This parameter specifies the T0 value.
betaValue ViReal64 This parameter specifies the beta value.

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
