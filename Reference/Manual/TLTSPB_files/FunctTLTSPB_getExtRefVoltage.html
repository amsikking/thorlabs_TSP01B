TLTSPB_getExtRefVoltage

ViStatus TLTSPB_getExtRefVoltage (ViSession instrumentHandle, double *referenceVoltage);

Purpose

This function returns the reference voltage of the external thermistor hw frontend if service mode has been enabled. Do not use in user applications.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
Output
Name Type Description
referenceVoltage double (passed by reference) Reference voltage of the external thermistor hardware frontend in Volt.

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
