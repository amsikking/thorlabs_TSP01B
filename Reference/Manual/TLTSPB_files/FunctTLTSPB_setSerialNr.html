TLTSPB_setSerialNr

ViStatus TLTSPB_setSerialNr (ViSession instrumentHandle, ViChar _VI_FAR serialNr[]);

Purpose

This function sets the serial number of the device if service mode has been enabled. Do not use in user applications.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
serialNr ViChar[] Production date of the device.

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
