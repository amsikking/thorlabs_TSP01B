TLTSPB_measHumidity

ViStatus TLTSPB_measHumidity (ViSession instrumentHandle, ViPReal64 humidity);

Purpose

This function is used to obtain relative humidity readings from the internal sensor. Same as TLTSPB_GetHumidityData().

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
Output
Name Type Description
humidity ViReal64 (passed by reference) This parameter returns the relative humidity in % r.H.

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
