TLTSPB_findRsrc

ViStatus TLTSPB_findRsrc (ViSession instrumentHandle, ViPUInt32 deviceCount);

Purpose

Gets the number of connected devices available in your system that can be
controlled with this driver.
instrHdl        This parameter is only needed for IVI compliant. Set to VI_NULL.
deviceCount     Receives the number of connected devices.
return          Status code.
/*-------------------- Prototype ---------------------*/
ViStatus TLTSPB_getDeviceCount (ViSession instrHdl, ViPUInt32 deviceCount);

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
Output
Name Type Description
deviceCount ViUInt32 (passed by reference) Returnes the number of connected devices you can control with this driver.

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
