TLTSPB_setTempSensOffset

ViStatus TLTSPB_setTempSensOffset (ViSession instrumentHandle, ViUInt16 channel, ViReal64 temperatureOffset);

Purpose

This function sets the sensor's absolute temperature offset in K
Note:
This function is valid for all temperature sensors only
Hint:
The offset will be added to the temperature value.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
channel ViUInt16 This parameter specifies the temperature channel.
Note:
This function is valid for all temperature sensors only (channels 1, 2 and 3)
temperatureOffset ViReal64 This parameter specifies the temperature offset in K.

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
